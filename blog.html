Bajrang Task Manager Full Stack Project

Overview

This project was built as part of the Bajrang Frontend Developer task. It is a full-stack web application where users can register, login and manage their personal tasks.

Why I built this

The goal was to demonstrate real-world frontend and backend integration using authentication, secure APIs, and a clean UI.

Features

  • User registration and login
  • JWT-based authentication
  • Add, view and delete personal tasks
  • Secure backend with FastAPI
  • React frontend using API integration

Tech Stack

  • Frontend: React + Vite
  • Backend: FastAPI
  • Database: MySQL
  • Authentication: JWT

Architecture

The frontend communicates with the FastAPI backend using REST APIs. JWT tokens are stored in localStorage and attached to every API request using Authorization headers.
